Simpl Cal in C

This program performs basic math:
- Add
- Subtract
- Multiply
- Divide

You enter how many numbers to process. The program repeats until you exit.

How to run
Download the release ZIP. Extract. Run the .exe.

How to compile
Windows:
gcc src/main.c -o simplcal.exe

Linux:
gcc src/main.c -o simplcal

Mac:
gcc src/main.c -o simplcal

Inputs needed
- Choose operation 1 to 4
- Enter number count
- Enter values one by one

Features
- Menu loop
- Uses scanf and printf
- Simple logic for students

Notes
Division uses float. It prints two decimals.
